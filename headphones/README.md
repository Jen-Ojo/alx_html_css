# ALX HTML & CSS — Headphones Project

*Repository:* alx_html_css  
*Directory:* headphones

This repo contains the step-by-step implementations for the ALX Headphones project. Each numbered task is a copy of the previous task with one new section or feature added.

---

## Files included

- 0-index.html, 0-styles.css — Task 0: Header + Hero
- 1-index.html, 1-styles.css — Task 1: Add "What we do..." section (uses icon font)
- 2-index.html, 2-styles.css — Task 2: Add "Our results" section
- 3-index.html, 3-styles.css — Task 3: Add Contact form
- 4-index.html, 4-styles.css — Task 4: Add Footer (full site)
- 6-index.html, 6-styles.css — Task 6: Draw pentagons with CSS (no image)
- 7-index.html, 7-styles.css — Task 7: Add animations to "What we do" and "Our results"
- 8-index.html, 8-styles.css, 8-script.js — Task 8: Add hamburger menu (JS-based)

---

## Project constraints implemented

- *Max content width:* 1000px centered.
- *Mobile layout breakpoint:* switch to mobile when screen width is *480px or less*.
- *Link hover/active color:* #FF6565.
- *Button hover/active:* opacity: 0.9.
- *Fonts:* Project uses system fonts to avoid external requests. If you want exact Figma fonts (Source Sans Pro, Spin-Cycle-OT), install locally or add Google Fonts links. See below.
- *Accessibility:* Focus outlines for keyboard navigation.
- *No JS* used for tasks 0–7.
- *Task 8* includes one small JS file to toggle the hamburger menu.
